TITLE & PURPOSE
----Scheduling Program----
The purpose of this program is to allow users to manage customers and appointments. Users are able to
add, modify, and delete customers and appointment using a simple GUI. The application also allows
users to view reports from the database.

HOW TO USE THE PROGRAM:
Open the program which will take the user to the login screen. Input a valid credential.

After logging in, the main view will have the appointment and customer table. Under the two tables are
buttons that users can utilize to manipulate the data, such as add, modify and delete.

If a user wants to add a customer or appointment, simply click the add button underneath the table which
will take the user to the add view. User can input valid information which will save to the database once
the saved button is clicked.

If a user wants to modify a customer or appointment, simply click the modify button underneath the table
which will take the user to the modify view. The modify view will have the existing data, and users will
be able to edit and save.

If a user wants to delete a customer or appointment, simply click the delete button underneath the table.
Upon clicking the delete button, an alert box will show to confirm the deletion.

On the bottom left corner, a view report button can be utilized to generate different reports.

A3F- The additional report that I created was appointment by type and location. This will gather all the type
of appointment and the different location and generate the total for each.
